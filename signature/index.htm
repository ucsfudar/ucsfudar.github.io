UCSF: The Campaign Email Signature

Instructions for Outlook desktop users (PC)

1. Select and copy a signature below

2. In Outlook (desktop), Open a new email. From the message tab select “Signature” then “Signatures…”

3. Select “new” at the bottom of the “Edit Signatures” box to create a new signature

4. Give your new signature a name

5. Paste the signature into the signature window

6. Select and type over the default information one line at a time, making sure to retain the type specifications

7. Click OK, then close the “Signatures” window

8. Insert your new signature into the email you are composing using the “Signatures” option, and consider setting as "Default"

Tips:

Please do not customize the colors (black), typeface (Arial), type size or logo graphics.
